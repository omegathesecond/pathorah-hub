## Instrumented Data Points (from codebase)

The platform already tracks the following events that can feed analytics:

- Path search requests (with depth, target type, credit cost)
- Introduction request creation, acceptance, and decline rates
- Credit consumption by user and subscription tier
- User registration and activation rates
- Company claim and verification events (tier 0 → 1 → 2 progression)
- Pathfinding timeout events (wall-clock cap exceeded)
